# provisioning
##Scripts to quickly provision fresh Linux installations.
- application_hardening.sh: Install MAC and sandboxing.
- apps.sh: List of some common apps I like.
- filesystems.sh: Modify the default permissions of some files.
- filtering.sh: Set up a firewall.
- kernel_modules.sh: Blacklist of some potentially dangerous modules.
- monitoring.sh: Install software to quickly audit a system.
- secure_grub.sh: Password-protect GRUB.
- spoof_macs.sh: Set up MAC addresses to be random to better resist enumeration.
